# ASU Faculty Salary Equity Analysis

An automated reporting system analyzing gender pay equity among 
Arizona State University faculty from 2016-2020.

## Project Overview
Built in response to growing concerns about gender pay disparities 
in university settings, this system automates the generation of 
annual salary equity reports by department and faculty rank.

## Components
- **utils.R** — Custom functions for data cleaning and visualization
- **salary-report.rmd** — Report template
- **batch.R** — Batch processing script to generate all yearly reports
- **Functions-Demo.rmd** — Function demonstrations

## Reports Generated
Annual salary equity reports for 2016, 2017, 2018, 2019, and 2020.

## Tools Used
R, R Markdown, dplyr, ggplot2
